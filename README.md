# [P2025‑08] Personal AI Wardrobe Assistant

## Requirements

The project aims to use multimodal large language models, such as Gemini and Qwen, to create a smart wardrobe assistant. Students will design a system that can recognize clothing items and style attributes from uploaded photos, provide outfit recommendations that adapt to different occasions and weather conditions, and allow users to virtually try on their selected outfits. The system should also be able to track and analyze the user's personal wardrobe history to support closet management. In addition, it should suggest new clothing items that complement the user's existing wardrobe while following current fashion trends.

This project will require students to have knowledge of web‑based development and machine learning, including skills in Python, HTML, CSS, JavaScript, SQL, API integration, image processing, and the practical use of LLM toolkits.



## Background

Traditional outfit selection can be time‑consuming, and online shopping is often limited by the inability to try on clothes virtually. Recent advances in multimodal large language models, computer vision, and recommendation systems make it possible to build intelligent wardrobe management tools. Such systems can provide personalized styling advice, try‑on experiences, and fashion insights, offering users both convenience and engagement.



## Outline

The objective of this project is to design a comprehensive AI wardrobe assistant application that allows users to digitize their personal closets and try on clothes virtually, with enhanced functionalities such as recognizing clothing style from uploaded images, providing recommendations based on different occasions and weather conditions, and suggesting new clothing items aligned with fashion trends. The software will also focus on building a digital clothing dataset, linking each clothing item with a corresponding description to support accurate search and recommendation.

The key requirements for this software are as follows:

- **Image Upload and Recognition**: The software must allow users to upload images of their clothing items, and with the help of multimodal large language models, recognize attributes such as category, color, material, and style.

- **2D Try‑On Algorithm**: The group must design or integrate an algorithm to enable virtual try‑on functionality.

- **Outfit Recommendation**: The system will provide outfit suggestions that adapt to user context, including weather conditions and the type of occasion, with explanations of why each recommendation is suitable.

- **Closet Management and History**: The application should maintain a record of wardrobe items and track usage history, helping users analyze what they wear most frequently and identify underused items.

- **Trend‑Aware Purchase Suggestions**: The software should suggest potential new items for the user to buy, considering gaps in the current wardrobe as well as current fashion trends.

- **Text‑ and Voice‑Based Interaction**: Users will be able to search or request outfits using descriptive keywords or natural language queries, with the LLM processing these instructions.

- **Efficient Browsing and Recommendations**: The application should allow for smooth browsing of wardrobe contents, with personalized suggestions based on user preferences and past interactions.

- **User Interface and Experience**: The application must offer a simple and user‑friendly interface, enabling seamless navigation between closet management, outfit recommendations, and try‑on functions.

- **Save and Share Functionality**: Users should be able to save outfit combinations they like and share them on social platforms.

  

## Tasks & Deliverables

- A functional web‑based application with an intuitive and user‑friendly interface.
- A developed digital wardrobe dataset with corresponding clothing attributes and descriptions.
- An intelligent clothing recognition and recommendation system powered by multimodal large language models.
- An efficient virtual try‑on feature integrated with the browsing and recommendation functions.
- A closet management module with personal history analysis and trend‑aware purchase suggestions.

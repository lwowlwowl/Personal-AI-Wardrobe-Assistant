
# 一、极简风（Minimalism）
核心特征：低对比、低纯度、线条干净、装饰最少、强调版型与面料质感。
代表颜色：黑白灰米色驼色藏蓝。
核心单品：直筒西装简约衬衫纯色针织直线剪裁大衣基础款长裤。
版型特点：合身或微宽松但结构清晰，无复杂层叠。
体形适配：适配全部体形，尤其适合矩形与梨形通过结构平衡比例。
场景适配：通勤正式日常高频使用。
禁忌：高饱和撞色复杂印花过多配饰。

# 二、通勤商务风（Business Casual / Formal）
核心特征：稳重克制、结构明确、强调专业感与整洁度。
代表颜色：黑灰藏蓝白低饱和蓝棕。
核心单品：西装外套衬衫西裤半身裙乐福鞋皮鞋。
版型特点：合身剪裁肩线清晰长度标准。
体形适配：所有体形，其中苹果形需弱化腰部、梨形需增强上半身结构。
场景适配：办公面试会议正式社交。
禁忌：运动单品牛仔破洞高纯度亮色。

# 三、韩系简约风（Korean Minimal）
核心特征：柔和低对比、版型宽松、氛围感强、年轻化。
代表颜色：米白浅灰雾蓝灰粉浅卡其。
核心单品：宽松衬衫针织开衫直筒裤半身长裙简洁风衣。
版型特点：微宽松垂坠感强调纵向线条。
体形适配：苹果形与矩形更友好，可弱化腰腹或增加层次。
场景适配：日常通勤校园社交。
禁忌：强对比撞色硬挺过度结构化西装。

# 四、日系自然风（Japanese Natural）
核心特征：宽松舒适、层次叠穿、低纯度自然色、强调生活感。
代表颜色：卡其橄榄绿棕灰米色靛蓝。
核心单品：宽松衬衫亚麻裤长裙针织马甲工装外套。
版型特点：明显宽松与多层叠穿结构。
体形适配：适合大多数体形，沙漏形需注意避免完全遮挡腰线。
场景适配：日常休闲旅行文艺活动。
禁忌：高饱和荧光色紧身剪裁。

# 五、法式优雅风（French Chic）
核心特征：自然精致、不过度用力、强调女性曲线与细节质感。
代表颜色：黑白藏蓝酒红奶油色。
核心单品：修身针织连衣裙高腰半裙风衣小高跟。
版型特点：合身收腰线条流畅不过紧。
体形适配：沙漏形最佳，梨形需平衡上下比例。
场景适配：约会社交轻正式场合。
禁忌：明显街头元素运动感过强单品。

# 六、美式休闲风（American Casual）
核心特征：舒适随性、功能导向、层次简单、运动元素常见。
代表颜色：牛仔蓝白灰红卡其。
核心单品：牛仔裤T恤卫衣运动鞋棒球帽夹克。
版型特点：宽松或直筒强调活动自由度。
体形适配：全部体形，其中苹果形适合直线剪裁上装。
场景适配：日常出行校园休闲周末活动。
禁忌：过度正式单品复杂装饰。

# 七、街头潮流风（Streetwear）
核心特征：高对比、大廓形、视觉冲击强、强调个性表达。
代表颜色：黑白高纯度亮色撞色组合。
核心单品：Oversize卫衣宽腿裤球鞋印花外套。
版型特点：明显宽松层次叠穿比例夸张。
体形适配：矩形与偏瘦体形更易呈现效果。
场景适配：潮流社交摄影音乐活动。
禁忌：严格正式场景使用。

# 八、运动机能风（Sport / Athleisure / Techwear）
核心特征：功能性面料、结构分割、强调运动与科技感。
代表颜色：黑灰荧光点缀军绿深蓝。
核心单品：运动套装冲锋衣束脚裤跑鞋功能背包。
版型特点：合体或机能宽松兼顾活动性。
体形适配：全部体形，苹果形注意避免腹部贴身。
场景适配：运动出行城市通勤户外。
禁忌：正式礼仪场景。

# 九、复古风（Vintage / Retro）
核心特征：历史时代元素、经典剪裁、色彩偏低明度。
代表颜色：棕墨绿酒红米黄复古蓝。
核心单品：高腰裤格纹西装针织开衫玛丽珍鞋。
版型特点：依据年代不同在修身与宽松间变化。
体形适配：沙漏与矩形更易表现经典比例。
场景适配：社交聚会摄影文化活动。
禁忌：现代强机能元素混用过多。

# 十、甜美少女风（Sweet / Cute）
核心特征：柔和色彩、曲线轮廓、装饰元素明显。
代表颜色：粉白浅紫奶油色浅蓝。
核心单品：荷叶边连衣裙蝴蝶结针织百褶裙玛丽珍鞋。
版型特点：收腰或A字轮廓强调可爱感。
体形适配：梨形与沙漏形较友好。
场景适配：约会休闲社交。
禁忌：高强度街头或严肃商务混搭。

# 十一、中性极简风（Gender Neutral）
核心特征：弱化性别曲线、结构简洁、色彩克制。
代表颜色：黑白灰深蓝军绿。
核心单品：宽松西装直筒裤基础衬衫皮鞋或球鞋。
版型特点：直线剪裁减少曲线强调。
体形适配：全部体形，尤其矩形自然契合。
场景适配：日常通勤艺术社交。
禁忌：过度甜美或曲线强化单品。

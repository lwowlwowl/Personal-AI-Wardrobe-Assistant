# 一、色彩基础结构
色彩由三要素构成：色相决定颜色类别如红橙黄绿蓝紫与黑白灰无彩色；明度决定轻重感，高明度更轻盈柔和，低明度更稳重正式；纯度决定鲜艳程度，高纯度活跃吸睛，低纯度克制高级。

# 二、主要配色关系
同色系为同一色相的明度或纯度变化，整体统一干净显高级，必须存在明度差避免单调；邻近色为色环相邻颜色组合，视觉柔和自然，适合日常休闲文艺与春夏穿搭；互补色为色环相对颜色，对比强烈视觉冲击明显，需遵循主辅比例约70/30并通过降低纯度获得高级感；无彩色搭配以黑白灰米色卡其藏蓝为核心，稳定安全适配所有年龄与场景，可加入低纯度彩色点缀提升层次。

# 三、明度与对比控制
高对比表现为明度差大或冷暖对比明显，视觉醒目，适用于年轻风格街头造型舞台摄影；低对比表现为明度接近与低纯度统一，适用于通勤极简成熟气质与正式环境。

# 四、季节性色彩策略
春季特征为高明度中高纯度偏暖色，代表色包括奶油白浅绿樱花粉浅黄；夏季为高明度低纯度冷色调，代表色雾蓝薰衣草紫灰粉浅灰；秋季为中低明度中等纯度偏暖，代表色焦糖橄榄绿驼色铁锈红；冬季为低明度高对比冷色，代表色纯黑宝蓝酒红冷白。

# 五、场景化配色规则
商务通勤以黑灰藏蓝为主色搭配白或米色辅色整体低对比强调稳重专业并避免高纯度亮色；日常休闲允许邻近色组合与中等纯度对比强调自然舒适与层次；正式礼仪以深色主导整体低纯度并严格控制亮色面积；时尚造型与摄影允许互补色高纯度与高对比用于视觉表达与风格强化。

# 六、禁忌规则
避免高纯度颜色大面积冲突、超过三种主色同时出现、明度与纯度完全一致导致单调、肤色偏暗却使用高亮荧光色、正式场合使用高饱和撞色。
